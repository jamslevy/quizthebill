  • There is no zero
  • Requires Python Imaging Library (PIL)

  • Uses free fonts from www.fonts4free.net

  • Font size: size of the text in font points, not pixels. Image size may vary, depending on chosen font
  • Text scale: characters are randomly scaled between 1.0 and value. 1.0 disables scaling
  • Text angle: characters are randomly rotated between -value and value. 0 disables rotating
